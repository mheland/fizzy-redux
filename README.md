A tasty blogging theme for Ghost :ghost:

- Replaced Tocify with TocBot, no jQuery dependency
- Uses Ghost 5 Native Discussions, removed GiTalk and Disqus
- No Math Library
- Removed Chinese fonts
- Carousel with Slider always active
